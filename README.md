# 🎧 Synced Media Experience App

A cross-platform (Android + Windows) application that enables two users to **watch or listen to media together in real-time**, with synced playback and interactive features.

---

## 🚀 Overview

This app allows long-distance users to share a **synchronized media experience** without streaming content between devices.

> Both users load the same media locally or via a common source, and the app syncs playback + interaction.

---

## ✨ Features

### 🎬 Synced Playback
- Play / Pause synchronization  
- Seek synchronization  
- Drift correction (auto re-sync)  
- Host or shared control  

---

### 📁 Local File Support
- Select media from device  
- File verification using hash (MD5/SHA)  
- Ensures both users have identical files  

---

### 💬 Real-Time Messaging
- Tap-based messaging  
- Floating on-screen messages  
- Context-aware (timestamp-based)  

---

### 🎙️ Voice Communication
- Peer-to-peer voice call  
- Mute / unmute  
- Low-latency audio  

---

### 🎨 Interactive Overlay System
- Tap highlights (focus points)  
- Freehand doodling  
- Circle annotations  
- Animated fade-out effects  

---

### 🎨 Personalization
- User-specific colors  
- UI themes (dark / neon / soft)  
- “Date mode” (minimal UI)  

---

### 👥 Presence System
- Shows connected users  
- Playback status (playing/paused)  
- Sync status indicator  

---

### 📜 Shared Queue *(Optional)*
- Add media to queue  
- Play-next system
